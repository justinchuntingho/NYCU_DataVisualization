# Data Visualization

Materials for the Data Visualization course offered by the Department of Communication and Technology, National Yang Ming Chiao Tung University.

Maintainer: Justin Chun-ting Ho

## Setup

We will be using [Jupyter Notebook Desktop](https://github.com/jupyterlab/jupyterlab-desktop) for the course, but you are free to use any IDE that can read and write `.ipynb`. If you are using Visual Studio Code, you could check out [this article](https://code.visualstudio.com/docs/datascience/jupyter-notebooks) for more information on how to use ipynb.

## Course Structure

This course is designed to teach basic Python programming, data wrangling, data visualization, and eventually building an interactive dashboard using Streamlit. Below I have listed the reading materials. Most of the readings are taken from [Computational Analysis of Communication](https://cssbook.net/), an open access book. For additional reference, I recommend following the free Python for Everybody course offered by Dr. Charles R. Severance (available as both [website](https://www.py4e.com/html3/) and [video](https://www.py4e.com/lessons)).
